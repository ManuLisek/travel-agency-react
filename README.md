## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ManuLisek/travel-agency-react.git
   ```
2. Open the root folder
   ```sh
   cd travel-agency-react
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run project
   ```sh
   npm start
   ```
